# PY_SEUXW

seuxw project in python.

## Menu

- `painter` 绘图模块
    - `bin` 二进制文件
        - `ttf` 字体文件
    - `constant` 常量
    - `database` SQL 建表语句
    - `util`
- `spider` 爬虫模块
    - `constant` 常量
    - `do` 爬虫调度器
    - `database` SQL 建表语句
    - `module`
        - `database` MySQL 操作器
        - `downloader` 下载器
        - `parser` 解析器
        - `redis` Redis 操作器
    - `util`
        - `common` 通用工具
            - date.py 日期操作工具
            - logger.py 日志工具
            - timeout.py 超时工具
        - `config` 配置工具
        - `database` MySQL 操作工具
        - `redis` Redis 操作工具
        - `web` 代理操作工具
